# KaPhysicsApp

Aplicación móvil multiplataforma enfocada a los alumnos de nivel preparatoria quienes cursan la materia de Física I. 

## Temas que se abordan en la app
- Cinemátgica:
  - Velocidad. 
  - Movimiento rectilíneo uniforme. 
  - Movimiento rectilíneo uniformemente acelerado. 
- Leyes de Newton:
  - Primera Ley.
  - Segunda Ley.
  - Tercera Ley.
- Trabajo y energía:
  - Trabajo y energía cinética.
  - Energía potencial.
  - Conservación de la energía.
